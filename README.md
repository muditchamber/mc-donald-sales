# McDonald's Sales Dashboard Analysis

## Project Overview

This project focuses on analyzing McDonald's sales data using Microsoft Excel. The dashboard was created to identify sales trends, customer ordering behavior, top-performing products, and revenue insights through interactive visualizations.

The project demonstrates practical data analytics skills including data cleaning, data visualization, KPI analysis, and business insight generation.

---

# Dashboard Preview

The dashboard includes:

* Total Revenue KPI
* Average Orders Per Day KPI
* Category-wise Revenue Analysis
* Item-wise Order Analysis
* Day-wise Sales Trend
* Monthly Category Performance
* Top Revenue-Generating Products
* Hourly Order Trend Analysis

---

# Objectives

The main objectives of this project are:

* Analyze sales performance across different food categories.
* Identify the most popular menu items.
* Study customer ordering patterns by time and day.
* Generate business insights for improving sales and operations.
* Create an interactive dashboard for decision-making.

---

# Key Insights

## Revenue Insights

* Total Revenue Generated: **61,626.29**
* Average Orders Per Day: **59.67**

## Category Analysis

* Burgers generated the highest revenue.
* Chicken and Pasta categories also performed well.
* Breakfast and Salad categories generated lower sales.

## Product Insights

Top ordered products:

* Quarter Pounder with Cheese
* Big Mac
* French Fries (Large)
* Chicken McNuggets (6-piece)

## Time-based Insights

* Peak sales occurred between **12 PM to 1 PM**.
* Evening sales increased around **5 PM to 6 PM**.
* Monday recorded the highest sales.
